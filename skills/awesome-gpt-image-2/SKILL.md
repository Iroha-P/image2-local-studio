# Awesome GPT Image 2 Local Skill

This project includes a small, local prompt skill inspired by the public
YouMind OpenLab `awesome-gpt-image-2` collection.

Use it by selecting a preset in the web UI, then replacing bracketed fields such
as `[产品名称]`, `[角色]`, or `[主题]` with the actual subject. The presets are
adapted summaries so the tool stays lightweight. The original repository remains
the source for the full community collection:

https://github.com/YouMind-OpenLab/awesome-gpt-image-2
