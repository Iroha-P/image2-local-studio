GPT Image 2 本地生图工具

Image2 Local Studio

一个可一键启动的本地图片生成工作台：支持官方原生尺寸、4K/自定义超分、图片墙、API 配置保存，以及可同步和翻译的中文 Prompt Skill 库。

查看 GitHub 仓库 使用说明

本地 API

API Key 保存在本机，浏览器只请求本地 Node 服务。运行 start.bat 后访问 localhost:8787。

尺寸与超分

按 OpenAI 官方尺寸生成，再用本地 sharp 保存为 2K、4K 或自定义尺寸。

中文 Skill

同步 YouMind 中文 README，并接入 EvoLinkAI 场景。英文场景可用保存的 OpenAI Key 一键翻译为中文。

图片墙

生成结果自动保存到本地图片墙，支持预览、下载和删除。
