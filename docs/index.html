为重复生图工作流做的本地控制台。

不是单张 prompt 表单，而是把参数、技能、预览、超分和文件管理放在一起。

API Control

OpenAI key 只保存在本地，支持模型、质量、背景、格式和安全参数配置。

Native + 4K

先按官方支持尺寸生成，再用本地 sharp 导出 2K、4K 或自定义尺寸。

Prompt Skills

同步 YouMind 和 EvoLinkAI 的案例，支持中文化、命名、删除、重置和编辑。

Image Wall

生成结果落到本地图片墙，保留预览、下载、删除、原始尺寸和导出尺寸。

01 / API SIZE官方尺寸先生成让请求保持在模型支持范围内，再进入本地后处理。
02 / SKILL EDITPrompt Skill 可改可重置同步仓库案例以后，在本地按自己的项目改字段。
03 / FINAL WALL超分后进入图片墙本地保存最终文件和元数据，方便再次下载或清理。

参数和图库不再分散。

左侧像控制台，右侧像工作板。生成前能看 skill 预览，生成后能直接整理图片。

1K
官方原生尺寸保持 API 调用稳定，不把不支持的尺寸硬塞给模型。
4K
本地最终导出超分和编码在本机完成，适合做封面、海报和素材归档。
中
中文 Skill 库英文案例可以翻译成中文，也可以手动调整为自己的调用模板。

从 Skill 到本地文件。

浏览器只负责操作体验，真正的 API 调用和文件保存都由本地 Node 服务处理。

01Choose skill

选一个场景，替换占位词，或者把英文案例翻译为中文。

02Generate native

用官方支持尺寸调用 GPT Image 2，保持请求稳定可复现。

03Upscale local

本地 sharp 负责最终 2K、4K 或自定义尺寸输出。

04Save wall

图片和元数据写入本地图片墙，方便预览、下载和删除。

把 GPT Image 2 变成本地创作桌面。

一键启动 Node 服务，浏览器打开控制台，开始生成和整理自己的图片库。
