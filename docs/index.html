本项目是本地工具，不是云端托管生图服务。GitHub Pages 只展示项目主页；真正的生图 API 需要在本机或支持 Node.js 的服务器上运行。

Features / 功能亮点

本地 API 代理

API Key 保存在本机，浏览器只访问 localhost:8787。

尺寸与超分

按 OpenAI 官方尺寸生成，再用 sharp 保存为 2K、4K 或自定义尺寸。

中文 Skill 库

同步 YouMind 中文 README，并接入 EvoLinkAI 场景；英文 Skill 可一键翻译。

图片墙

生成结果自动保存到本地，支持预览、下载和删除。

Workflow / 工作流

  1. 双击 start.bat 启动本地服务。
  2. 保存 OpenAI API Key，选择官方尺寸和最终导出尺寸。
  3. 选择或编辑 Prompt Skill，必要时翻译为中文。
  4. 生成图片，结果进入本地图片墙。
